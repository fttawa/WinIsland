# 什么是 WinIsland？

WinIsland 是一个为 Windows 桌面打造的现代灵动岛实现。受 iOS 启发，它提供了一个集中的枢纽，用于媒体控制、系统状态以及各种工具，同时拥有流畅且不唐突的设计。

## 核心特性

- **动态缩放**：在迷你和展开视图之间无缝过渡。
- **媒体控制**：查看专辑封面、歌曲信息以及播放状态。
- **频谱可视化**：实时的音频频谱分析器。
- **轻量级**：使用 Rust 编写，占用极低的系统资源。

## 为什么选择 WinIsland？

与其他“灵动岛”克隆产品不同，WinIsland 专注于成为一个实用工具，而不仅仅是一个视觉噱头。它与 Windows SMTC（系统媒体传输控制）深度集成，以提供准确、实时的信息。
